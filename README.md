# Token-Economy-System
This Token Economy System is designed to support individuals with Autism Spectrum Disorder (ASD) by giving therapists, teachers, caregivers, and clinicians a simple and customizable way to reinforce positive behaviors. The project lets you create individual client profiles and quickly open any profile to manage their token board. You can choose an emoji token from the built-in options or use your own custom emoji, making it easy to tailor the experience to what motivates each client. Rewards can be created and linked to the client, and tokens can be given as positive behaviors occur. When a client earns enough tokens, the reward can be claimed with a single action, and the token board can be reset to start fresh. The goal of this system is to make reinforcement clear, consistent, and personalized while staying easy for professionals to manage. By offering flexibility, visual support, and a straightforward workflow, the system helps keep clients motivated, encourages the repetition of positive behaviors, and makes sessions smoother and more engaging for everyone involved.
